## 🚫 Patches Excluídos (e por quê)

### Bugados (v1.30.0)
| Patch | Problema |
|-------|----------|
| SponsorBlock | Toast congela a UI ([#1555](https://github.com/MorpheApp/morphe-patches/issues/1555)) |
| Return YouTube Dislike | Congela app 5-10s ao abrir vídeo ([#1451](https://github.com/MorpheApp/morphe-patches/issues/1451)) |
| Sanitize sharing links | Quebrado ([#1602](https://github.com/MorpheApp/morphe-patches/issues/1602)) |
| Open system share sheet | Não funciona ([#1597](https://github.com/MorpheApp/morphe-patches/issues/1602)) |
| Track crossfade (Music) | Quebra casting ([#1549](https://github.com/MorpheApp/morphe-patches/issues/1549)) |
| Alternative thumbnails | API instável |

### Niche / Desnecessário
| Patch | Por que não |
|-------|-------------|
| Bypass URL redirects | Ninguém percebe a diferença |
| Copy video URL | Share já faz isso |
| Loop video | Quase ninguém usa |
| Miniplayer | O padrão funciona bem |
| Hide player flyout menu | Menu que ninguém abre |
| Hide video action buttons | Remove botões úteis |
| Spoof device dimensions | Só alguns dispositivos |
| Open Shorts in regular player | Niche |
| Change miniplayer color | Cosmético |
| Hide category bar (Music) | Ganho mínimo |
| Hide buttons (Music) | Pode esconder útil |
| Navigation bar (Music) | Ganho mínimo |
